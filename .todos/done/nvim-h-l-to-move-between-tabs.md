# nvim H L to move between tabs

**Status:** Done
**Date:** 2026-02-08

## Details
- Implemented H and L keybinds to navigate between tabs in Neovim
- H moves to previous tab
- L moves to next tab
- Centralized in whichkey.lua with proper leader keybindings

## Notes
- Integrated with modular config architecture
- Non-leader keybinds stored separately in init.lua
