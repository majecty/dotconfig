# emacs 설정

## 설정 파일 설치하기

[init.el](../init.el) 파일을 soft link로 ~/.emacs.d/init.el에 링크를 걸거나 기본 설정파일에서 해당 파일을 load합니다.

### windows의 경우 다음 링크를 참고하여 설정파일 위치를 찾습니다.

<https://www.gnu.org/software/emacs/manual/html_node/emacs/Windows-HOME.html>
